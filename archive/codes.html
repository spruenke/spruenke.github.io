All available codes are written in R or R-related C/C++. For usage one needs at least R, however RStudio is recommended as IDE. Both are Open Source.

Overview

Package for nonparametric hypothesis testing on cluster data (alpha version / pre-release)

The package provides several utilities and functions to compute hypothesis tests on cluster data in the several sample case.

  • Wald-test
  • ANOVA-type test
  • Multiple Contrast Test Procedure
  • Distribution Functions
  • Nonparametric Relative Effects
  • Covariance-Matrix of Relative Effects

LaTeX-package and templates

The package provides several templates for title pages (thesis and working papers), a template for exercise sheets and a package (.sty) providing environments and many useful commands for statistics and stochastics.

  • Environments for lemma, theorem, definition
  • Commands for Calculus, Linear Algebra, Stochastics, Statistics
  • Title page for Bachelor/Master thesis
  • Title page for working papers
  • Template for Exercise sheets
